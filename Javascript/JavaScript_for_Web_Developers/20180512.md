## 4장. 변수와 스코프, 메모리

### 4.1 원시 값과 참조 값

원시값: Undefined, Null, 불리언, 숫자, 문자열
참조값: 메모리에 저장된 객체

4.1.1. 동적 

~~~javascript
var.person = new Object();
person.name = "Nicholas";
alert(person.name);
~~~
참조값을 다룰때는 언제든 프로퍼티와 메서드를 추가하고 바꾸고 삭제할 수 있음

~~~javascript
var name = "Nicholas";
name.age = 27;
alert(name.age);
~~~
원시 값에는 프로퍼티가 없지만 추가하려해도 에러가 생기진 않음

**동적으로 프로퍼티를 추가할 수 있는 값은 참조 값 뿐**
